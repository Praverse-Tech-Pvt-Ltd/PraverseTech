# **App Name**: PraverseAI

## Core Features:

- AI-Powered Compliance Automation: Automate FDA 483/Warning Letter responses, SOP drafting, and DI audits using AI.
- HealthMate Kiosk: AI-powered healthcare kiosk for multimodal triage and EHR integration.
- Contact Form with reCAPTCHA: Contact form with reCAPTCHA Enterprise for spam protection; sends data to Firestore and email via Cloud Function.
- Blog and Case Studies: MDX-powered blog with tags, reading time, and share buttons.
- Newsletter Signup: Collects newsletter subscribers and stores data in Firestore.
- Admin Panel: Basic CRUD interface to add and edit blog posts and testimonials.
- Dark/Light Mode Toggle: Enables users to switch between dark and light UI modes.

## Style Guidelines:

- Primary color: Deep navy (#0B132B) for a premium and trustworthy feel.
- Accent color: Electric blue (#4F46E5) for highlighting key elements and CTAs.
- Secondary accent: Teal accents (#06B6D4) to add vibrancy.
- Background color: Warm white (#F8FAFC) to ensure a clean, modern aesthetic.
- Headings font: 'Inter' sans-serif for a modern and clean look.
- Body font: 'Inter' sans-serif for readability and consistency.
- Code font: 'JetBrains Mono' monospace for displaying code snippets.
- Use 'lucide-react' icons for a consistent and modern look.
- Implement a clean and premium grid-based layout with generous whitespace and glassmorphism accents.
- Use framer-motion for smooth transitions and subtle parallax effects in the hero section.
